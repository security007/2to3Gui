# 2to3Gui

Tool to convert python2 to python3 GUI version

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requirement module.

```bash
pip install -r requirements.txt
```

## Usage

```bash 
python main.py
```
